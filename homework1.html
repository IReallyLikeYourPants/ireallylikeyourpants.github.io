T1.
Online we can find various definitions of Statistics. The two most common one are:
- It is the discipline that concerns the collection, organization, analysis, interpretation, and presentation of data
- It is a science (systematic body of knowledge) which collects and uses data
Some sites refer to Statistics as a branch of Math that collects, analyzes and interprets data, but this definition is not correct as Statistics is not the same things as Math, not even a branch, because they only share the theory of probability.
We can visualize Statistics as a science that touch differents disciplines like Chemistry, Physics, Sociology, Biology and interacts with them.
To perform a statistical search, it’s conventional to begin with a population of interest the so colled "statistical population" ("statistical model") that it’s made by the statistical units (all objects that we want to analyze).
Statistics can be used for example to create a model to predict certain reality behaviour and it's used in different fields such as economy, medicine, sociology, science.

T2.
Dataset refers to any organized collection of data, typically it is represented in a table way. It lists variables for each of the attributes (column) and for each member of the dataset (row) so called record/unit.
A unit is one member of a set of entities being studied and it has one (univariate statistics) or more attributes (multivariate statistics). All the units are called set of statistical units.
Attributes are specific qualities, characteristics or features that can be shared by multiple subjects like body weight, hair colour, the temperature outside.
Variables are the building block of statistic and they consist of information coming from observations or measurement like values from body weight, hair colour, the temperature outside, etc and they can be created, analyzed, manipulated and processed by computer programs.
It is also necessary to decide the scale of measurements or the precision of the measure of each attribute. There are generally two measures: quantitative measure (meters, centimeters) and qualitative one (tall, short or medium).
Datasets are stored tyipically in an excel file or in a DBMS.
Datasets and Statistics can be used for example to create a model to predict certain reality behaviour, to better understand reality and minimize risks resulting from randomness.

T3.
There are different applications in Cybersecurity that use Statistics, examples of this are:
- Cyber risk, the probability of a vulnerability being exploited thanks to statistical analysis and probabilistic programming
- Understanding whether a site is malicious or benign
- Understanding whether a pdf is malignant or benign
- Analysing the most frequent attacks used on websites
- Attacking methods like brute force that uses methods to increment their chances of success
- Easiest way to take control of a network
- Honeypots that collect information about attackers, we can use them to improve the security of the systems
- Firewalls that detect abnormal behaviour from a host thanks to probability
- Frequency analysis, discover plaintext from the frequency of letters of the ciphertext

TA1.
Main differences between C# and VB.NET:

C# VB.NET
C# uses C-based syntax VB.NET uses simple English in its syntax
C# is a case-sensitive language VB.NET is not a case-sensitive language
C# supports structured error handling VB.NET supports structured and unstructured error handling
C# is more restrictive VB.NET uses implicit casting
C# uses two equal to compare VB.NET uses a single equal to compare
C# uses camelCase as a naming convention VB.NET uses PascalCase as a naming convention
C# uses the + symbol to concatenate strings VB.NET uses the & symbol to concatenate strings
C# uses the normal keyword "null" VB.NET uses the keyword "Nothing"
Events are not possible in C# Events are automatically bound
To declare a variable, "declarators" is used in C# Private, Public, Friend, Protected, Static 1, Shared, Dim’ is used in VB.NET
C# uses "this" to refer to current object VB.NET uses "Me" to refer to current object
C# has pointers VB.NET has not pointers
In C# we can overload a function or a method without writing anything In VB.NET we can overload with the keyword Overloads
To create an Object in C# we have to use the keyword new To create an Object in VB.NET we use New CreateObject()
C# is preferred by expert programmers VB.NET is most user-friendly
C# code is not automatically formatted VB.NET code automatically gets formatted
C# support hiding VB.NET doesn't support hiding
Shadowing is not supported in C# Shadowing is supported in VB.NET
Object Clean is done by Destructor Object Clean is done by Finalize Method
In C# declare a class by using the Class keyword In VB.NET declare a class by using Class "implementation" keyword
The object can be created using New The object is created using New and Create Object()
In C# refer to a base class by using the base keyword In VB.NET refer to a base class by using the MyBase keyword
Basically used for the development of desktop-based applications Basically used for the development of Microsoft-based applications
In C#, garbage collection is not automatic In VB.NET, garbage collection is automatic
To retrieve characters from a string, "[]" is used in C# To retrieve characters from a string, "GetChar Function" is used in VB.NET
Each statement ends with a semicolon ( ;) Each statement does not terminate with a semicolon
Each function starts with curly bracket ( {) and ends with another curly bracket ( }) In VB.NET no curly bracket. When use a if condition, we have to use english words ( If <condition> Then … End if)
Variable definition: <varibale type> name of variable (ex, int var) Variable definition: <Dim> name of variable <As> <variable type> (ex, Dim var As Integer)
